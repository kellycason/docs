# Changelog

All notable changes to the Power Pages Code Site Agent Skill are documented here.

This project uses [Semantic Versioning](https://semver.org/).

## 1.0.0 - 2026-07-22

### Added

- Installable Agent Skill package with progressive reference loading.
- End-to-end Power Pages SPA scaffolding and validation guide.
- React/Vite, Dataverse Web API, table-permission, authentication, PAC CLI, and
  deployment patterns.
- SharePoint document management through Server Logic and Microsoft Graph.
- Least-privilege `Sites.Selected`, binary upload-session, IDOR protection,
  download, inline PDF/image preview, and cache-propagation guidance.
- Workspace and global installers for PowerShell and Bash.

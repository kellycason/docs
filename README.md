## Agent Skills

| Skill | Browse | Install |
| --- | --- | --- |
| Power Pages Code Site | [Open package](skills/power-pages-code-site/) | [Download latest ZIP](https://github.com/kellycason/docs/releases/latest/download/power-pages-code-site.zip) |

The Power Pages skill helps coding agents build, secure, deploy, debug, and
validate React/Vite Power Pages SPA code sites, including Dataverse, Server
Logic, and SharePoint document management.

Install it in one workspace at:

```text
.github/skills/power-pages-code-site/
```

Or install it globally at:

```text
~/.copilot/skills/power-pages-code-site/
```

See the [skill README](skills/power-pages-code-site/README.md) for installers,
version pinning, updates, and example prompts.

Copilot Studio: Enterprise Agent Platform

An overview of how organizations can use Copilot Studio to build, govern, and publish AI agents across the Microsoft ecosystem.

Overview

Start with business outcomes. Copilot Studio brings agent authoring, enterprise data, workflow actions, publishing channels, and governance into the Microsoft platforms many organizations already use.

Core platform capabilities

AreaCopilot Studio capability
Productivity integrationPublish agents to familiar channels such as Microsoft 365 Copilot, Teams, SharePoint, websites, and custom apps so users can access help where they already work.
Enterprise dataGround agents in Microsoft 365 content, SharePoint, Dataverse, Dynamics 365, Microsoft Graph-connected sources, and other systems through Copilot and Power Platform connectors.
Security & complianceUse Microsoft Entra, Microsoft Purview, tenant controls, environment policies, role-based access, auditing, and data loss prevention as part of the broader Microsoft governance model.
Agent governanceManage environments, connectors, sharing, publishing workflows, admin approvals, telemetry, and lifecycle controls through Power Platform and Microsoft 365 administration experiences.
Business process automationLet agents retrieve data and take action through Power Platform connectors, Power Automate flows, custom connectors, APIs, Dynamics 365, and line-of-business systems.
Low-code + pro-codeBusiness teams can start with natural-language and visual authoring, while developers extend scenarios with custom connectors, APIs, Azure services, and application lifecycle practices.

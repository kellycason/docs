Microsoft Copilot Studio

Pricing & Licensing Options

How Copilot Studio is licensed, the ways to buy, and how Copilot Credits are consumed — so you can pick the right model to start.

Bottom line

Start from your usage pattern: pay-as-you-go for pilots and variable workloads, and the Copilot Credit Pre-Purchase Plan (P3) when annual scale justifies an upfront commitment.

Ways to buy

Microsoft 365 Copilot

Per-user subscription

Best when employees are already licensed and agents extend Microsoft 365, Teams, and SharePoint. Qualifying employee-facing agent usage is included under fair-use limits; Work IQ APIs still consume Copilot Credits.

Copilot Studio pay-as-you-go

Usage-based · no commitment

Post-pay only for the credits agents consume each month via an Azure-linked billing policy. Ideal for pilots, seasonal demand, and business continuity with no upfront purchase.

Copilot Credit Pre-Purchase Plan (P3)

Annual commitment

Buy a one-year pool of credits, drawn down automatically as agents run. Overage continues on pay-as-you-go; unused credits expire at the end of the term.

What you need to start

  • Maker access — a Copilot Studio user license (no cost), the author role, a Microsoft 365 Copilot license, or a trial.
  • A billing path — pay-as-you-go or the Copilot Credit Pre-Purchase Plan (P3), or qualifying Microsoft 365 Copilot rights.
  • Governance (recommended) — set up environment strategy, DLP, spend policies, alerts, and per-agent limits as you scale.
  • A usage estimate — sized with the Copilot Credit usage estimator.

How credits are consumed

Copilot Credits are the common, tenant-pooled currency for agents. Consumption scales with how an agent retrieves knowledge, reasons, runs workflows, calls AI tools, takes actions, and handles voice.

Tenant-pooledCredits are shared across the tenant and allocated per environment.
No commitmentPay-as-you-go bills only for what agents use each month.
One-year termPre-Purchase Plan (P3) credits are prepaid and expire at term end.

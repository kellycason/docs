Know the difference

Capacity allocation assigns a budget, Azure budgets send alerts, and per-agent limits can stop usage. These are separate controls.

What each control actually does

Environment allocation

Assigns a planned share of prepaid tenant capacity. It isn't an exact hard stop.

Draw from tenant

Lets an environment use available shared prepaid credits after its allocation.

Bill to PAYG

Sends applicable usage or overage to the linked Azure subscription.

Azure budget

Sends actual or forecasted cost alerts. It doesn't stop agents or PAYG consumption.

Per-agent monthly limit

Can notify admins and automatically turn off that agent at its limit, where available.

Disable or disconnect

Stops the agent or removes the PAYG path; another valid capacity path might be required.

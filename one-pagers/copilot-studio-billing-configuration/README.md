# Copilot Studio Billing Configuration One-Pager

A GCC-focused, customer-facing comparison of Copilot Studio prepaid, pay-as-you-go, spillover, overage, alert, and hard-stop configurations.

## Files

- `index.html` - the billing configuration guide. Renders as a responsive web page and prints to a one-page letter PDF.
- `pdf/copilot-studio-billing-configuration.pdf` - exported PDF linked from the page.
- `source-notes/copilot-studio-billing-configuration-source-notes.md` - verified billing facts, official Microsoft sources, and GCC availability guardrails.
